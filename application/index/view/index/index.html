关于我们

  1. 介绍 - 上海陆加奉装饰材料有限公司已经成立十三年，主营各种进口高端窗帘，墙纸，地毯，家具，灯具。目前年销售额在两千万左右。
  2. 优势 - 本公司主要代理意大利及美国的多个顶级品牌的窗帘，墙纸，地毯，家具，灯具。在中国市场具有一定的知名度。 合作项目基本都是国内著名的酒店及豪宅项目。
  3. 配置 - 本公司在用人上都是使用具有十几年高端项目操作经验的顶尖人才，加工以及施工人员也都是国内一流水准。

供应商

陆加奉优选来自意大利以及美国的众多知名高端家私品牌，例如Vescome,Omexco等。

为了给消费者提供更高端，更有品味的家私产品，陆加奉根据自身从业十多年的经验以及与各品牌的密切接触了解，严选了在世界范围内拥有较高知名度的品牌。

正是对消费者的认真负责，让陆加奉在国内的高端家私行业走过了十多年。在这十多年中，陆加奉以优秀的品质以及服务，赢得了国内许多高端酒店，豪宅品牌的认可。
